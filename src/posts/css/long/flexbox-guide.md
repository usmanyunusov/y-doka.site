–°–≤–æ–π—Å—Ç–≤–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª—è–µ—Ç —Å–≤–æ–±–æ–¥–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –ø–æ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –æ—Å–∏ –º–µ–∂–¥—É —Å—Ç—Ä–æ–∫–∞–º–∏ —Ñ–ª–µ–∫—Å-—ç–ª–µ–º–µ–Ω—Ç–æ–≤. –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —É –≤–∞—Å 11 —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –Ω–∞ 3 —Å—Ç—Ä–æ–∫–∞—Ö. –ï—Å–ª–∏ —Ä–∞–∑–º–µ—Ä —Ä–æ–¥–∏—Ç–µ–ª—è –ø–æ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –æ—Å–∏ –ø–æ–∑–≤–æ–ª—è–µ—Ç, —Ç–æ –ø—Ä–∏ –ø–æ–º–æ—â–∏ `align-content` –º–æ–∂–Ω–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª—è—Ç—å —Å—Ç—Ä–æ—á–∫–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, —Å–≤–µ—Ä—Ö—É, —Å–Ω–∏–∑—É, –ø–æ —Ü–µ–Ω—Ç—Ä—É –∏–ª–∏ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ.

–ù–µ –∏–º–µ–µ—Ç –≤–∏–¥–∏–º–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç—ã —Ä–∞—Å–ø–æ–ª–∞–≥–∞—é—Ç—Å—è –≤ –æ–¥–Ω—É —Å—Ç—Ä–æ–∫—É.

–í–æ–∑–º–æ–∂–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è:

- `stretch` (–∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é) ‚Äî —Å—Ç—Ä–æ–∫–∏ —Ä–∞—Å—Ç—è–≥–∏–≤–∞—é—Ç—Å—è –æ–¥–∏–Ω–∞–∫–æ–≤–æ —Ç–∞–∫, —á—Ç–æ–±—ã –∑–∞–Ω—è—Ç—å –≤—Å—ë –¥–æ—Å—Ç—É–ø–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ —Ä–æ–¥–∏—Ç–µ–ª—è.

- `flex-end`¬†/¬†`end` ‚Äî –≤—Å–µ —Å—Ç—Ä–æ–∫–∏ —Ä–∞—Å–ø–æ–ª–∞–≥–∞—é—Ç—Å—è —É –∫–æ–Ω—Ü–∞ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –æ—Å–∏. `end` —Å–º–æ—Ç—Ä–∏—Ç –Ω–∞ —Ä–µ–∂–∏–º —á—Ç–µ–Ω–∏—è —Ç–µ–∫—É—â–µ–≥–æ —è–∑—ã–∫–∞.
- `center` ‚Äî —Å—Ç—Ä–æ–∫–∏ –≤—ã—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ø–æ —Ü–µ–Ω—Ç—Ä—É —Ä–æ–¥–∏—Ç–µ–ª—è.
- `space-between` ‚Äî¬†–ø–µ—Ä–≤–∞—è —Å—Ç—Ä–æ–∫–∞ –ø—Ä–∏–∂–∏–º–∞–µ—Ç—Å—è –∫ –Ω–∞—á–∞–ª—É –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –æ—Å–∏, –ø–æ—Å–ª–µ–¥–Ω—è—è ‚Äî –∫ –∫–æ–Ω—Ü—É –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –æ—Å–∏, –∞ –æ—Å—Ç–∞–ª—å–Ω—ã–µ —Ä–∞—Å–ø–æ–ª–∞–≥–∞—é—Ç—Å—è —Ç–∞–∫, —á—Ç–æ–±—ã —Å–≤–æ–±–æ–¥–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –±—ã–ª–æ –ø–æ–¥–µ–ª–µ–Ω–æ –Ω–∞ –æ—Ç—Å—Ç—É–ø—ã –º–µ–∂–¥—É –Ω–∏–º–∏ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ.
- `space-around` ‚Äî¬†–æ—Ç—Å—Ç—É–ø—ã —É –∫–∞–∂–¥–æ–π —Å—Ç—Ä–æ–∫–∏ —Ä–∞–≤–Ω–æ–∑–Ω–∞—á–Ω—ã –æ—Ç—Å—Ç—É–ø–∞–º —É –ª—é–±–æ–π –¥—Ä—É–≥–æ–π —Å—Ç—Ä–æ–∫–∏.
- `space-evenly` ‚Äî –æ—Ç—Å—Ç—É–ø—ã –º–µ–∂–¥—É —Å—Ç—Ä–æ–∫–∞–º–∏ –∏ –æ—Ç –∫—Ä–∞—ë–≤ —Ä–æ–¥–∏—Ç–µ–ª—è –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ.

–ü—Ä–∏ –ø–æ–º–æ—â–∏ —Å–≤–æ–π—Å—Ç–≤–∞ `order` –º–æ–∂–Ω–æ –º–µ–Ω—è—Ç—å –ø–æ—Ä—è–¥–æ–∫ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è —Ñ–ª–µ–∫—Å-—ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤–Ω—É—Ç—Ä–∏ —Ñ–ª–µ–∫—Å-–∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞.

–ü–æ —É–º–æ–ª—á–∞–Ω–∏—é —ç–ª–µ–º–µ–Ω—Ç—ã –æ—Ç–æ–±—Ä–∞–∂–∞—é—Ç—Å—è –≤ —Ç–æ–º –ø–æ—Ä—è–¥–∫–µ, –≤ –∫–æ—Ç–æ—Ä–æ–º –æ–Ω–∏ —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω—ã –≤ —Ä–∞–∑–º–µ—Ç–∫–µ. –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –∑–Ω–∞—á–µ–Ω–∏–µ —Å–≤–æ–π—Å—Ç–≤–∞ `order` —Ä–∞–≤–Ω–æ 0. –ù–æ —ç—Ç–æ —Å–≤–æ–π—Å—Ç–≤–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ—Å—Ç–∞–≤–ª—è—Ç—å –∏—Ö –º–µ—Å—Ç–∞–º–∏.

–ó–Ω–∞—á–µ–Ω–∏–µ –∑–∞–¥–∞—ë—Ç—Å—è –≤ –≤–∏–¥–µ —Ü–µ–ª–æ–≥–æ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –∏–ª–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–≥–æ —á–∏—Å–ª–∞. –≠–ª–µ–º–µ–Ω—Ç—ã –≤—Å—Ç–∞—é—Ç –ø–æ –≤–æ–∑—Ä–∞—Å—Ç–∞—é—â–µ–π.

### `flex-grow`

```css
.container {
  display: flex;
}

.item {
  flex-grow: 1;

```

–≠—Ç–æ —Å–≤–æ–π—Å—Ç–≤–æ —É–∫–∞–∑—ã–≤–∞–µ—Ç, –º–æ–∂–µ—Ç –ª–∏ –∏ –Ω–∞ —Å–∫–æ–ª—å–∫–æ –≤—ã—Ä–∞—Å—Ç–∞—Ç—å —Ñ–ª–µ–∫—Å-—ç–ª–µ–º–µ–Ω—Ç –ø—Ä–∏ –Ω–∞–ª–∏—á–∏–∏ —Å–≤–æ–±–æ–¥–Ω–æ–≥–æ –º–µ—Å—Ç–∞. –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –∑–Ω–∞—á–µ–Ω–∏–µ —Ä–∞–≤–Ω–æ 0. –ó–Ω–∞—á–µ–Ω–∏–µ–º –º–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±–æ–µ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ (–≤–∫–ª—é—á–∞—è 0).

–ï—Å–ª–∏ —É –≤—Å–µ—Ö —Ñ–ª–µ–∫—Å-—ç–ª–µ–º–µ–Ω—Ç–æ–≤ –±—É–¥–µ—Ç –ø—Ä–æ–ø–∏—Å–∞–Ω–æ `flex-grow: 1`, —Ç–æ —Å–≤–æ–±–æ–¥–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –≤ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–µ –±—É–¥–µ—Ç —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–æ –º–µ–∂–¥—É –Ω–∏–º–∏ –≤—Å–µ–º–∏.

–ï—Å–ª–∏ –ø—Ä–∏ —ç—Ç–æ–º –æ–¥–Ω–æ–º—É –∏–∑ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –º—ã –∑–∞–¥–∞–¥–∏–º `flex-grow: 2`, —Ç–æ –æ–Ω –ø–æ—Å—Ç–∞—Ä–∞–µ—Ç—Å—è _–æ—Ç–∂–∞—Ç—å_ —Å–µ–±–µ –≤ –¥–≤–∞ —Ä–∞–∑–∞ –±–æ–ª—å—à–µ —Å–≤–æ–±–æ–¥–Ω–æ–≥–æ –º–µ—Å—Ç–∞, —á–µ–º –µ–≥–æ —Å–æ—Å–µ–¥–∏.

```css

  display: flex;
}

  flex-shrink: 3;
```

–°–≤–æ–π—Å—Ç–≤–æ `flex-shrink` –ø–æ–ª–Ω–æ—Å—Ç—å—é –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ —Å–≤–æ–π—Å—Ç–≤—É `flex-grow`. –ï—Å–ª–∏ –≤ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–µ –Ω–µ —Ö–≤–∞—Ç–∞–µ—Ç –º–µ—Å—Ç–∞ –¥–ª—è —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏—è –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –±–µ–∑ –∏–∑–º–µ–Ω–µ–Ω–∏—è —Ä–∞–∑–º–µ—Ä–æ–≤, —Ç–æ —Å–≤–æ–π—Å—Ç–≤–æ `flex-shrink` —É–∫–∞–∑—ã–≤–∞–µ—Ç –≤ –∫–∞–∫–∏—Ö –ø—Ä–æ–ø–æ—Ä—Ü–∏—è—Ö —ç–ª–µ–º–µ–Ω—Ç –±—É–¥–µ—Ç —É–º–µ–Ω—å—à–∞—Ç—å—Å—è.

–ß–µ–º –±–æ–ª—å—à–µ –∑–Ω–∞—á–µ–Ω–∏–µ —É —ç—Ç–æ–≥–æ —Å–≤–æ–π—Å—Ç–≤–∞, —Ç–µ–º –±—ã—Å—Ç—Ä–µ–µ —ç–ª–µ–º–µ–Ω—Ç –±—É–¥–µ—Ç —Å–∂–∏–º–∞—Ç—å—Å—è –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å —Å–æ—Å–µ–¥—è–º–∏, –∏–º–µ—é—â–∏–º–∏ –º–µ–Ω—å—à–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

–ó–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é ‚Äî 1. –ó–Ω–∞—á–µ–Ω–∏–µ–º –º–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±–æ–µ —Ü–µ–ª–æ–µ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ (–≤–∫–ª—é—á–∞—è 0).

–î–≤–∞ –ø—Ä–µ–¥—ã–¥—É—â–∏—Ö —Å–≤–æ–π—Å—Ç–≤–∞ —Ä–∞–±–æ—Ç–∞—é—Ç —Å –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–º —Ä–∞–∑–¥–µ–ª–µ–Ω–∏–µ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞, –Ω–µ —Å –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–º–∏ —Ä–∞–∑–º–µ—Ä–∞–º–∏. –û–Ω–∏ –¥–æ–≤–æ–ª—å–Ω–æ –Ω–µ–ø—Ä–æ—Å—Ç—ã–µ, –¥–∞–∂–µ –æ–ø—ã—Ç–Ω—ã–π —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫ –Ω–µ –≤—Å–µ–≥–¥–∞ –∑–Ω–∞–µ—Ç, –∫–∞–∫ –æ–Ω–∏ –≤ —Ç–æ—á–Ω–æ—Å—Ç–∏ —Ä–∞–±–æ—Ç–∞—é—Ç. –ó–∞–≥–ª—è–Ω–∏ –≤ –∫–æ–Ω–µ—Ü —Å—Ç–∞—Ç—å–∏, –µ—Å–ª–∏ —Ö–æ—á–µ—à—å –ø–æ–¥—Ä–æ–±–Ω–µ–µ –ø–æ—á–∏—Ç–∞—Ç—å –æ –∫–∞–∂–¥–æ–º –∏–∑ –Ω–∏—Ö.

### `flex-basis`

```css
.container {
  display: flex;
}

.item {
  flex-basis: 250px;

```

–°–≤–æ–π—Å—Ç–≤–æ `flex-basis` —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ —Ä–∞–∑–º–µ—Ä —ç–ª–µ–º–µ–Ω—Ç–∞ –¥–æ —Ç–æ–≥–æ, –∫–∞–∫ —Å–≤–æ–±–æ–¥–Ω–æ–µ –º–µ—Å—Ç–æ –±—É–¥–µ—Ç —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–æ (—Å–º. [flex-grow](/posts/css/long/flexbox-guide/#flex-grow)).

–ó–Ω–∞—á–µ–Ω–∏–µ–º –º–æ–∂–µ—Ç –±—ã—Ç—å —Ä–∞–∑–º–µ—Ä –≤ –ª—é–±—ã—Ö –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω—ã—Ö –∏–ª–∏ –∞–±—Å–æ–ª—é—Ç–Ω—ã—Ö –µ–¥–∏–Ω–∏—Ü–∞—Ö: `20rem`, `5vw`, `250px`. –ê —Ç–∞–∫–∂–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ `auto` (–∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é). –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ –ø—Ä–∏ —Ä–∞—Å—á—ë—Ç–µ —Ä–∞–∑–º–µ—Ä–∞ —ç–ª–µ–º–µ–Ω—Ç–∞ –±—É–¥—É—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å—Å—è –≤–æ –≤–Ω–∏–º–∞–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è —Å–≤–æ–π—Å—Ç–≤ `width` / `max-width` / `min-width` –∏–ª–∏ –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ –≤—ã—Å–æ—Ç—ã. –ó–∞–≤–∏—Å–∏—Ç –æ—Ç —Ç–æ–≥–æ, –≤ –∫–∞–∫–æ–º –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏–∏ –∏–¥—ë—Ç –æ—Å–Ω–æ–≤–Ω–∞—è –æ—Å—å.

–ï—Å–ª–∏ –Ω–∏–∫–∞–∫–∏–µ —Ä–∞–∑–º–µ—Ä—ã –Ω–µ –∑–∞–¥–∞–Ω—ã, –∞ —Å–≤–æ–π—Å—Ç–≤—É `flex-basis` —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ `auto`, —Ç–æ —ç–ª–µ–º–µ–Ω—Ç –∑–∞–Ω–∏–º–∞–µ—Ç —Å—Ç–æ–ª—å–∫–æ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞, —Å–∫–æ–ª—å–∫–æ –Ω—É–∂–Ω–æ –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è –∫–æ–Ω—Ç–µ–Ω—Ç–∞.

### `flex`

```css
.container {
  display: flex;
}

.item {
  flex: 1 1 auto;
}
```

–°–≤–æ–π—Å—Ç–≤–æ-—à–æ—Ä—Ç–∫–∞—Ç, —Å –ø–æ–º–æ—â—å—é –∫–æ—Ç–æ—Ä–æ–≥–æ –º–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ —Ç—Ä—ë—Ö —Å–≤–æ–π—Å—Ç–≤ –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ: `flex-grow`, `flex-shrink` –∏ `flex-basis`. –ü–µ—Ä–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ —è–≤–ª—è–µ—Ç—Å—è –æ–±—è–∑–∞—Ç–µ–ª—å–Ω—ã–º, –æ—Å—Ç–∞–ª—å–Ω—ã–µ –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã.

–ó–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é: `0 1 auto`, —á—Ç–æ —Ä–∞—Å—à–∏—Ñ—Ä–æ–≤—ã–≤–∞–µ—Ç—Å—è –∫–∞–∫ `flex-grow: 0`, `flex-shrink: 1`, `flex-basis: auto`.

–í–æ–∑–º–æ–∂–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è:

```css

/* –û–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ, —á–∏—Å–ª–æ –±–µ–∑ –µ–¥–∏–Ω–∏—Ü: flex-grow */

/* –û–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ, —à–∏—Ä–∏–Ω–∞/–≤—ã—Å–æ—Ç–∞: flex-basis */
```
